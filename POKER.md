# 德州扑克 功能变更记录

> 每次新增或修改功能后，在此文件末尾追加记录。

---

## 基础功能（初始版本）

- 多人实时对战，最多 10 人桌
- 完整德州扑克规则：翻牌 / 转牌 / 河牌 / 摊牌
- 房主控制开始每一手，非房主需点击「准备」
- 自定义盲注和初始筹码
- 支持 Heads-up（2人对局）特殊规则
- 重复昵称自动追加 ID 后缀

---

## 动画与音效

- 下注筹码飞向底池动画
- 动作气泡（弃牌 / 过牌 / 跟注 / 加注 / All-in / 盲注）
- 摊牌逐个揭示开牌动画，赢家座位高亮发光
- 赢家音效（上行琶音）
- 程序化背景音乐（Jazz Lounge，WebAudio 合成）

---

## Docker 部署支持

- 添加 `Dockerfile` 和 `docker-compose.yml`
- 支持阿里云 ECS + DaoCloud 镜像源拉取 `node:20-alpine`
- SQLite 数据库通过 Docker Volume `poker-db` 持久化

---

## AI 实时分析（MiniMax）

- 每手发牌后自动调用 MiniMax AI（模型优先级：M2.5-highspeed → M2.7 → M2.5）
- 展示三张独立卡片：**胜率**、**建议行动**、**原因**
- 翻牌 / 转牌 / 河牌后自动刷新分析
- 原因使用通俗中文，禁止英文和专业术语
- 每轮 AI 思考时间上限 60 秒（回合超时同步延长至 60 秒）
- MiniMax API Key 通过 `.env` 环境变量读取，不提交到仓库

---

## TTS 语音播报（MiniMax speech-2.8-hd）

- 玩家每个动作实时语音播报：弃牌 / 过牌 / 跟注 / 加注 / 全押 / 盲注
- 每种动作配对应情境语气：
  - 弃牌：sad（低沉沮丧，语速 0.9）
  - 过牌：calm（平静，语速 1.0）
  - 跟注：calm（淡定，语速 1.05）
  - 加注：happy（自信爽快，语速 1.1）
  - 全押：angry（强硬激烈，语速 1.2）
  - 盲注：calm（例行，语速 1.0）
- 每手结束播报获胜玩家姓名和赢得筹码（happy 语气，语速 0.95）
- 服务器启动时预生成动作音频并缓存到磁盘，避免重复调用 API

## 体验与功能全面优化

### 超时紧急警告
- 倒计时剩余 ≤ 10 秒时，计时器变红并脉冲闪烁，提醒玩家尽快行动

### AI 预计算
- 新轮到自己行动时，若 AI 状态为 idle 或 error，自动触发分析请求（安全兜底）
- 避免因错过触发时机导致 AI 面板一直空白

### TTS 播报时机优化
- 获胜播报改为 `speakAfter()`：立即请求音频，等摊牌动画完成后再播放
- 消除音频请求延迟导致的空白等待，播报时机与动画精准对齐

### 边池（Side Pot）显示
- All-in 场景下实时计算并展示主池 / 边池分布
- 替换单一底池数字，清晰呈现多池结构

### 断线重连
- 服务端缓存每手结算结果（`lastHandSummary`）
- 玩家重连后自动补发上一手结算，确保不错过结果
- 仅对已在房间内的玩家生效，新加入玩家不受影响

### 手牌历史回放
- 历史记录列表新增「详情」按钮
- 弹窗展示：最终公共牌、各方手牌及牌型、获胜者、分阶段行动记录
- 行动记录按 PREFLOP / FLOP / TURN / RIVER 分组，颜色区分弃牌 / 加注 / All-in
